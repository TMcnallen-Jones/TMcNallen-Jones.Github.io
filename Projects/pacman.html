Pac-Man C++ Project

C++ SFML SQL GitHub

This was made in 15 weeks using C++ and SFML and was my first game made without a game engine, It helped me get a better understanding, since I had to make the state machine myself. I always enjoyed Pac-Man as a kid so I really enjoyed researching the specifications of pacman and coming up with my own implementation. I even made a highscore system using sql to create, update and display a database and a level generation function that will create a render of a level based on a map given to it in ASCII. This allows for the addition of more levels with relative ease.

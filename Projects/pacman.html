Pac-Man C++ project

This is a project I made in the first year at university. It was made in C++ using SFML and was my first game made without
a game engine, It helped me get a better understanding, since I had to make the state machine myself. I always enjoyed
pacman as a kid so I really enjoyed researching the specifications of pacman and coming up with my own implementation.

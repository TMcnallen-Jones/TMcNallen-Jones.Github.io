Unity Group Project

This is a project that was made in unity, with multiple people working on it at the same time. I was the programmer, so used C# to make the movement, combat, enemy AI, health, menu and audio systems. We were given five weeks to make a game together. Every week we would meet up for a scrum meeting, where I would lead the meetings to discuss what we have done and what we need to be doing next. This really helped improve my communication, teamwork and leadership skills. It was great for improving my skills and it was also a lot of fun!
